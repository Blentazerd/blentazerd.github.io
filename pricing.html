LibreOffice

Гайд как установить LibreOffice из официального репозитория Arch Linux

Для установки свежей версии используйте следующую команду

sudo pacman -S libreoffice-fresh

Для установки стабильной версии используйте следующую команду

sudo pacman -S libreoffice-still

После установки одной из версию LibreOffice вы можете приступать к работе с приложением, однако я рекомендую поставить два полезных пакета

Для установки русскоязычного пакета вы можете использовать команду для своей версии LibreOffice

Команда для свежей версии (fresh)

sudo pacman -S libreoffice-fresh-ru

Команда для стабильной версии (fresh)

sudo pacman -S libreoffice-still-ru

Также рекомендую установить пакет Languagetool чтобы проверять текст на орфографические ошибки

sudo pacman -S languagetool

Теперь вы можете переходить к запуску LibreOffice

Этой командой можно запустить LibreOffice через терминал

libreoffice

"Скопируйте команду и вставьте в терминал"
